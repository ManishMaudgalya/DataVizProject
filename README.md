# Euro 2024 Analytics DataViz

## Overview
This project is a Streamlit-based data visualization application for Euro 2024 football (soccer) analytics. It provides interactive dashboards and visualizations to analyze match events, player passing networks, pitch control, and Expected Threat (xT) models.

## Project Structure
- `app.py`: The main Streamlit application script containing the dashboard layout and visualizations.
- `data_engine.py`: Handles raw data ingestion, cleaning, and processing.
- `pass_network.py`: Functions to generate and visualize player passing networks.
- `pitch_control.py`: Calculates and maps spatial pitch control across the field.
- `xT_model.py`: Evaluates and models Expected Threat (xT) from match events.
- `requirements.txt`: List of Python dependencies required to run the application.
